## 需求文档与原型设计说明书

### 1. 项目概述

本平台旨在基于 **BettaFish** 多智能体框架，通过对 TikTok、Temu、SHEIN 及社交论坛的多模态数据进行抓取与深度分析，为跨境电商卖家提供“短期动向监测 + 长期趋势分析 + 交互式知识沉淀”的一站式选品方案。

---

### 2. 核心功能模块设计

#### 2.1 模块一：短期动向预览 (T+7 极速看板)

- **功能描述：** 针对用户输入的关键词（如“Portable Coffee Maker”），快速抓取过去 7-10 天的公开数据。
    
- **核心指标：**
    - **热度指数：** 基于点赞、转发、评论增长率计算的 Hotness Score。
    - **情绪分布：** 消费者的正负面评价倾向。
        
- **交互组件：**
    - **词云图 (Word Cloud)：** 提取视频标题、评论区的高频核心卖点或吐槽点。
    - **动向曲线图：** 展示过去一周内相关话题在不同平台的曝光走势。

#### 2.2 模块二：深度趋势报告 (长期分析)

- **功能描述：** 在用户确认选品方向后，启动长周期（3-6 个月）的全网深度扫描。
- **输出内容：**
    - **竞品对比：** 自动对比 Temu、SHEIN、Amazon 上的同类 SKU 价格与评分。
    - **生命周期预测：** 利用 Agent 研判该品类处于“爆发期”、“平台期”还是“衰退期”。
    - **PDF 导出：** 自动渲染生成包含数据图表、Agent 专家建议、供应链风险提示的专业报告。

#### 2.3 模块三：AI 选品智库 (RAG 交互)

- **功能描述：** 将 PDF 报告及抓取的素材入库，转化为可对话的知识资产。
- **核心特性：**
    - **智能追问：** 支持如“根据报告，这款产品的理想定价区间是多少？”等具体问题的回答。
    - **溯源功能：** 所有的 AI 回答必须附带原报告的 Chunk（片段）或原始网页链接。

---

### 3. 技术架构设计 (推荐方案)

- **前端 (Vue.js 3)：** 采用 **Naive UI** 或 **Element Plus**，视觉风格采用你偏好的“复古墨水/纸质”质感，强化报告阅读的仪式感。
- **后端 (Django / FastAPI)：** * 使用 **Celery + Redis** 处理耗时的 Agent 爬取和报告生成任务。
    - **Agent 引擎：** 继承 BettaFish 的 Media/Query Agent。
- **AI 层 (RAG)：** * **Vector DB:** Milvus 或 ChromaDB。
    - **LLM:** 采用 GPT-4o 或 Claude 3.5 Sonnet 进行深度分析，部署 Llama-3-8B 负责基础清洗。

---

### 4. 原型界面初步构思

|**页面**|**主要功能布局**|
|---|---|
|**仪表盘 (Home)**|一个极简的搜索框，支持“关键词 + 目标平台”输入。下方展示“今日全球热点趋势”。|
|**快速预览页 (Quick Preview)**|左侧为词云图，中间为热度曲线，右侧为 TikTok 热门视频抽样预览。底部提供【发起深度分析】按钮。|
|**报告生成页 (Report Hub)**|列表展示已生成的 PDF。点击进入后，左侧显示 PDF 阅读器，右侧显示 AI 对话窗口。|
|**配置中心 (Settings)**|配置抓取频率、关注的竞争对手站点、以及各平台的 API 密钥或代理。|

---

### 5. 项目进度建议 (Roadmap)

1. **Phase 1 (MVP):** 实现 TikTok 指定标签的爬取 + Vue 词云图展示 + 基础 PDF 生成。
2. **Phase 2 (AI Up):** 接入 RAG 系统，实现针对 PDF 报告的对话功能。
3. **Phase 3 (Full Integration):** 引入多平台（Temu/SHEIN）横向对比功能，优化 Agent 的“选品决策逻辑”。
