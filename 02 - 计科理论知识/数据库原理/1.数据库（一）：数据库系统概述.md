# 一、数据库系统基本概念
## 1.数据

- **数据**（Ddta）：数据库中存储的基本对象，其定义为**描述事物的符号**。
- 除了数值以外，文字、图片和音视频等也是数据。
## 2.数据库

- **数据库**（Database）：长期存储在计算机内有组织、可共享的**大量数据的集合**。主要特点如下：
	- 数据库中的数据**按照一定的数据模型组织、描述和存储**；
	- 具有**较小的数据冗余**；
	- 较高的**数据独立性和可拓展性**。
# 3.数据库管理系统

- 是位于用户（软件开发者）与操作系统之间的数据库管理软件，是和操作系统一样的计算机基础软件，一类大型复杂的软件系统。主要有以下几个功能：
	1. 数据定义功能：提供数据定义语言（data definition language,DDL），方便用户定义数据对象的组成与结构
	2. 数据组织、存储和管理功能
	3. 数据的操纵功能
	4. 数据库的事务管理和运行管理功能
	5. 数据库的简历和维护功能
	6. 其他功能：与操作系统、其他应用的通信、异构数据库之间的互操作
## 4.数据库系统

- 指引入数据库之后的计算机系统，由数据库、数据库管理系统（及应用开发工具）、应用系统和**数据库管理员**组成的存储、管理、处理和维护数据的系统。
- 建立、使用和维护数据库的专业人员称为数据库管理员，也是数据库系统中的重要组成部分之一。
