# 码驿随想 项目文档

## 项目概述

**项目名称**：码驿随想
**项目域名**：www.mysx.top
**技术栈**：Vue 3 + Django 5.2.8 + MySQL + Cloudflare R2
**代码统计**：
- 前端代码：约 25,804 行
- 后端代码：约 8,018 行
- **总计：约 33,822 行**

## 一、系统需求分析

### 1.1 功能需求清单

| 功能模块 | 需求描述 |
|----------|----------|
| **用户认证** | 用户注册（邮箱验证码）、登录（图形验证码）、密码重置、个人信息管理、头像上传、账号注销 |
| **文章系统** | 文章发布、分类标签管理、Markdown编辑、点赞收藏、评论互动、搜索功能、专栏组织 |
| **影集管理** | 照片上传、分类组织、展示浏览、位置记录、日期管理 |
| **项目管理** | 项目展示、图标/图片、标签分类、下载链接/网址链接 |
| **评论系统** | 文章评论、评论审核、回复功能 |
| **通知系统** | 管理员发布通知、通知展示、按日期查询 |
| **统计系统** | 网站访问量统计、文章浏览量、点赞收藏量、仪表盘可视化 |
| **图床功能** | 图片上传、自动压缩转WebP、Cloudflare R2存储、CDN加速 |
| **知识图谱** | 文章关系可视化、ECharts力导向图 |
| **SEO优化** | 动态Sitemap生成、页面元信息、结构化数据 |

### 1.2 非功能需求

| 需求类型 | 具体要求 |
|----------|----------|
| **安全性** | JWT认证、验证码防机器人、邮箱验证、用户状态中间件、密码加密 |
| **性能** | 图片自动压缩、CDN加速、分页查询、缓存策略 |
| **可用性** | 响应式设计、深浅主题切换、加载动画、错误处理 |
| **可维护性** | Docker容器化、环境配置分离、日志记录、代码规范 |
