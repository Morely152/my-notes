# 二、项目化简

### ① 文件增删

删除多余的文件，保留或创建以下必要的文件，得到最简的Vue项目：

![](20260106095757081.png)

这些文件的作用分别如下所述：

- **node_modules**：依赖包下载之后的存放目录
- **public**：存放全局的静态文件，如网页的图标、图片等素材
- **src**
	- **assets**：一般是存放代码引用的静态文件，比如：css、js、img
	- **components**：存放 vue 的组件（可复用的网页部件，如导航栏、评论区等）
	- **router**：定义路由文件的目录，相当于整个网站的导航
	- **views**：存放 vue 网页文件的目录，一个文件相当于一个页面
	- **App.vue**：vue 页面全局的入口，是所有 vue 文件的父级
	- **main.js**：代码的配置文件，引入第三方的组件或者我们自己定义的一些组件、css、js 等
- **index.html**：vue编译成网页才能在浏览器渲染
- **jsconfig.json**：内部配置文件
- **package.json**：定义依赖库的文件
- **package-lock.json**：下载依赖的时候将版本锁定
- **vite.config.js**：全局的配置文件

# 二、初步框架搭建

## 1.设置网页标签名和图标

在 `index.html` 中的 `<head>` 部分，设置网页全局标题和图标。

如果想要实现页面不同标题，在路由中配置元数据并且配置页面加载时将网页标签名修改为路由中的元数据即可。

## 2.配置全局样式

在 `assets` 文件夹中新建 `css/global.css`，然后在 `global.css` 中编写全局的css样式：

```css
:root {
  /* filter: grayscale(100%); */    /* 应用此样式，可以让全网站变成灰色 */
  --text-color: #333;               /* 可以直接在CSS中使用var(--text-color)实现统一配色 */
}

* {
    box-sizing: border-box;
}

body {
    margin: 0;
    padding: 0;
    color: #333;
    font-size: 14px;
}

a {
    text-decoration: none;
}
```

写好之后，记得在 `main.js` 中引入，以应用到所有的网页中：

```js
import './assets/css/global.css'
```

## 3.添加404页面处理

用户输错网址或者跳转到已经删除的页面时，会由于没有内容而一片空白。我们需要进行提示和引导：

### ① 路由配置404重定向

新建 `views/404.vue`，然后修改 `router/index.js` ：

```js
import { createRouter, createWebHistory } from 'vue-router'

const router = createRouter({
  history: createWebHistory(import.meta.env.BASE_URL),
  routes: [
    { path: '/notFound', component: () => import('@/views/404.vue') },   // 定义404页面的路由
    { path: '/:pathMatch(.*)', redirect: '/notFound' }  // 如果访问路由中没有定义的页面，重定向到404页面
  ]
})

export default router
```

### ② 404页面编写

``` vue
<template>
  <div style="height: 100vh; display: flex; align-items: center; justify-content: center">
    <div style="width: 50%">
      <img style="width: 100%" src="@/assets/imgs/404.jpg" alt="">
      <div style="text-align: center; padding: 20px 0; font-size: 20px;">
	      <a style="color: #3741fb" href="/">返回主页</a>   <!-- href="/"，点击后跳转回主页 -->
	  </div>
    </div>
  </div>
</template>
```

# 三、Element-Plus集成

[Element-Plus](https://element-plus.org/zh-CN/component/overview) 是一套适用于 Vue 3 的前端UI框架，提供了丰富精美的Web常用控件，可以用来方便地构建Web前端页面:

![](20260106102042842.png)

## 1.安装Element-Plus依赖

在终端执行 `npm i element-plus -S`，等待安装完成即可。

查看 `package.json`，在 `dependences` 中出现 `"element-plus: "^2.x.x"` 即表示安装成功。

使用 `npm uninstall element-plus` 即可卸载依赖。

## 2.引入Element Plus

```js
// main.js
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import zhCn from 'element-plus/es/locale/lang/zh-cn'

app.use(ElementPlus, {
  locale: zhCn,
})  
```

## 3.全局使用Element Plus的图标

![](20260106113420260.png)

首先安装依赖：

```shell
npm install @element-plus/icons-vue
```

引入依赖：

```js
// main.js
import * as ElementPlusIconsVue from '@element-plus/icons-vue'

for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
  app.component(key, component)
}
```

可在这些标签中使用图标：el-icon、el-input、el-button

- el-input：在输入框组件里面使用图标，**仍需要单独导入图标**
- el-icon、el-button：按钮或者图标组件里面，**不需要单独导入图标**
