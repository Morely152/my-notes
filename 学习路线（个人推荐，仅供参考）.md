# 一、前备知识

此部分知识按需学习，如果已经了解可以直接跳过。
## 1.Web开发（HTML+CSS+JS）

- 速成版：[3小时前端入门教程（HTML+CSS+JS）](https://www.bilibili.com/video/BV1BT4y1W7Aw)
- 也可以参考一下[大前端系列教程](https://www.mysx.top/columns?column=1)，这是我学习Web时写的一些笔记，与学校所发的教材内容同步，学习前面的HTML+CSS+JS三节即可，JQuery现在使用不多。需要重点学习CSS中的Flex布局和Grid布局，这两种元素的布局方式比较常用。

# 2.Git（版本控制工具）

- 推荐学习：[【GeekHour】一小时Git教程](https://www.bilibili.com/video/BV1HM411377j/)，学会在VS Code中使用拓展工具拉取/推送代码到远程仓库即可，分支等高级操作可按需学习。

## 3.Java

（新手阶段Java的学习不用太深入，在后期学习Spring Boot时，可以得到补充和强化）

- 推荐学习：[Java教程-廖雪峰的官方网站](https://liaoxuefeng.com/books/java/introduction/index.html)，先快速了解其中“快速入门”、“面向对象编程”、“异常处理”这几个部分即可，其余内容可以在学习Spring Boot及后面的专业课学习时再进行补充。

--- 
# 二、Vue+Spring Boot前后端开发

- 推荐课程：[【带小白做毕设】2025](https://www.bilibili.com/video/BV1HaKnePEBj)，包含了Vue（前端页面）+Spring Boot（后端服务）+MySQL（数据库增删改查）的整套内容。

项目级的开发实战可能会有些难度，一定要跟着老师的讲解和操作来完整地把代码敲出来，学习之后会有很大的收获。

--- 
# 三、Vibe Coding（氛围编程）

进入AI时代，应用开发的很多工作都已经可以让AI完成了。这里也推荐一下B站UP主“程序员鱼皮”的[鱼皮 Vibe Coding 零基础入门教程](https://ai.codefather.cn/library/2010994846520700929)，可以系统化学习一下如何使用自然语言来指挥Claude Code、Trae、Cursor等AI工具完成开发任务。
