# 打包上传服务器步骤

## 1.本地开发环境配置修改

- mysx_back/mysx_back/settings.py:
	- **line 23**：is_deploy = False

- mysx_front/src/services/api.js:
	- **line 6**：export const API_BASE_URL = 'http://192.168.56.1:8000/api';

```bash
cd mysx_front
npm run dev

cd mysx_back
 python .\manage.py runserver 0.0.0.0:8000
```

## 2.服务器文件部署

0. 停止本地前后端服务器，备份数据库
1. 恢复上面两个文件，后端选用部署模式，前端请求目标改为api路径
2. 压缩mysx_back，cd到前端项目，先install然后`npm run bulid`，完成之后压缩dist
3. 登陆宝塔，停止前后端站点
4. mysx_back.zip上传解压到/www/wwwroot/mysx-django
5. dist.zip上传解压到/www/wwwroot/mysx-vue
6. 开启前后端站点
7. 开启代理，登录cloudflare控制台，域 - 缓存 -配置 - 清除缓存 - 清除所有内容
