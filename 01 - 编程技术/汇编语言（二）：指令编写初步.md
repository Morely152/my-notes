# 一、基本的086汇编程序结构

类似于C语言的main方法，

```c
#include <stdio.h>
int main(void) {
	// 主程序逻辑
	return 0;
}
```

8086汇编程序一般也有这样一个固定的模式：

``` asm
; 1.数据段：存放程序运行时需要用到的一些数据值
DATA SEGMENT

DATA ENDS

; 2.代码段：核心执行逻辑
CODE SEGMENT
    ASSUME CS:CODE, DS:DATA  ; 声明段寄存器关联
    
; 标记汇编程序入口的标志
START:
    ; 3. 初始化数据段（必须步骤，让CPU找到数据）
    MOV AX, DATA
    MOV DS, AX

    ; 4. 编写汇编程序的逻辑指令
    
    
    ; 5. 程序正常退出（必须要交还操作权限给Dos，否则程序会卡死）
    MOV AH, 4CH      ; AH=4CH：程序退出功能号
    INT 21H          ; 执行退出

CODE ENDS

; 6.程序入口标记
END START
```

0.可以发现汇编语言的程序是按段定义的，使用`XXX SEGMENT -- XXX ENDS`这样的指令来定义一个段；
1.数据段类似于C语言程序一开始的`int a = 3;  char b;`这样的定义，预定好程序运行所需的数据值；
2.代码段类似于`int main(void)`方法，在这里写程序的指令逻辑；
3.注意这里不能直接写`MOV DS， DATA`，因为8086 CPU 不允许直接把立即数/段地址送入段寄存器` (DS/ES/SS/CS)`，没有设计这样的数据传输电路（也就自然没有这样的指令），因此必须使用AX进行中转，`DATA`中存放的是段地址，如`1000h`；
5.常用的DOS功能号如下所示；
6.`END`表示程序结束，`START`告诉编译器程序从`START`标志处开始执行。

## 1. DOS常用功能号

使用DOS功能的统一格式：

``` asm
MOV AH, 功能号    ; 设置要做什么
INT 21H           ; 执行DOS中断
```

就像是先在纸条上写好要做什么，然后交给DOS等它去办理一样。
### ① **01H —— 键盘输入 1 个字符（有回显）**

- 作用：等待用户按一个键，读到 **AL** 中（逻辑写死了，一定是放到AL中）
- 入口：`AH=01H`
- 出口：**AL = 输入字符的 ASCII 码**

``` asm
MOV AH, 01H   ; 功能号：读入单个字符
INT 21H       ; 执行后 AL 里就是输入的字符
```

### ② **02H —— 屏幕输出 1 个字符

- 作用：在光标位置打印一个字符
- 入口：`AH=02H`，**DL = 要打印的字符**

``` asm
MOV DL, 'A'   ; 要打印的字符（也可以用 31h 表示 1）
MOV AH, 02H   ; 功能号：打印字符
INT 21H
```

### ③ **08H —— 键盘无回显输入**

- 作用：输入但屏幕不显示，适合用于输入密码时保护隐私

``` asm
MOV AH, 08H
INT 21H      ; 结果存在 AL 中
```

### ④ **09H —— 屏幕输出字符串**

- 作用：输出一串字符
- 规则：字符串必须以 **`$` 结束**
- 入口：`AH=09H`，**DS:DX = 字符串首地址**

``` asm
; 数据段定义：
STRING DB 'Hello, Assembly!$'  ; 必须以 $ 结尾

LEA DX, STRING    ; DX = 字符串偏移地址

MOV AH, 09H       ; 功能号：打印字符串
INT 21H
```

### ⑤ **0AH —— 键盘输入字符串**

- 作用：读取一行输入（带回车结束）
- 入口：`AH=0AH`，**DS:DX = 输入缓冲区首地址**

``` asm
LEA DX, BUF   ; DX指向输入缓冲区

MOV AH, 0AH   ; 功能号：读入字符串
INT 21H
```

### ⑥ **4CH —— 退出程序**

- 作用：结束程序，返回 DOS
- 入口：`AH=4CH`

``` asm
MOV AH, 4CH   ; 功能号：退出程序
INT 21H       ; 程序结束
```


### 速记表：
- 01H：键盘输入1个字符到AL中
- 02H：屏幕输出DL中的1个字符
- 08H：键盘无回显输入（记忆秘诀：8位数密码 -> 08H适合用来输入密码）
- 09H：屏幕输出字符串
- 0AH：键盘输入字符串
- 4CH：退出程序
